# 快速上手

## 安装

```bash
npm install wsvaio-ui
```

## 使用

```vue
<script lang="ts" setup>
// 按需引入
import { WsLabelValue } from "wsvaio-ui";
</script>

<template>
  <ws-label-value label="label">value</ws-label-value>
</template>
```
